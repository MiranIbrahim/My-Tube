Discuss the scope of var, let, and const.

Scope of Var:

-The scope of a var variable is a functional scope.
-It can be updated and re-declared into the scope.

Scope of Let:

-The scope of a let variable is a block scope.
-It can be updated but cannot be re-declared into the scope.

Scope of Const:

-The scope of a const variable is also a block scope.
-It cannot be updated or re-declared into the scope.

Use Cases of Null and Undefined

null:

  • null is used when we want to explicitly indicate that a variable or object property intentionally has no value or is empty.
  • It is often used to reset or clear a reference to an object, indicating that it should no longer point to any specific value or object.
  • We can use null as a placeholder when we expect a value to be assigned later in the code.

What do you mean by REST API?

An API, or application programming interface, is a set of rules that define how applications or devices can connect to and communicate with each other. A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style. For this reason, REST APIs are sometimes referred to RESTful APIs.

HTTP Methods in REST APIs

GET :

Retrieve data from the server.

POST

Create a new resource on the server.

PUT

Update an existing resource on the server (usually by replacing it).

PATCH

Partially update an existing resource.

DELETE

Remove a resource from the server.
